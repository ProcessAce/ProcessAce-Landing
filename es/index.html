Cómo funciona

Cuatro sencillos pasos desde la evidencia cruda hasta la documentación de procesos aplicable.

01

Subir Evidencia

Sube documentos de texto como POEs (SOPs), notas de reuniones o correos que describan tus procesos.

02

Análisis con IA

ProcessAce utiliza el LLM de tu elección para extraer pasos, actores, sistemas y relaciones de la evidencia.

03

Generar Artefactos

Obtén diagramas BPMN profesionales, tablas SIPOC/RACI y documentación narrativa generados automáticamente.

04

Editar y Exportar

Refina los artefactos con el editor interactivo. Exporta como XML, PNG, CSV, Markdown o PDF.

En marcha en minutos

Docker Compose es todo lo que necesitas. Usa tu propia API key de LLM.

1. Clona el repositorio
git clone https://github.com/jgleiser/ProcessAce.git
cd ProcessAce
2. Configura el entorno
cp .env.example .env
# Edita .env y configura ENCRYPTION_KEY
3. Ejecuta con Docker Compose
docker compose up -d --build
4. Abre la interfaz web
# Navega a http://localhost:3000
# Registra tu primera cuenta (será Administrador)
# Configura el proveedor de LLM en App Settings

Requisitos previos

  • Docker y Docker Compose
  • Una API key de LLM (OpenAI, Google GenAI, o Anthropic)
  • Una cadena Hexadecimal de 32 bytes (para el cifrado seguro de las API keys)
