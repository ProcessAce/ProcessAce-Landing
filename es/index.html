Todo lo que necesitas para descubrir procesos

Desde evidencia en bruto hasta documentación profesional, totalmente automatizado.

Ingesta de Evidencia

Sube POEs, notas, correos y documentos de texto. Ingesta nativa de archivos de Audio y Video con un flujo de trabajo de transcripción primero y revisión humana en el bucle.

Análisis Multi-LLM

Elige entre OpenAI, Google Gemini, Anthropic Claude, o ejecuta modelos locales totalmente aislados a través de nuestra integración de primera clase con Ollama.

Genera Artefactos

Genera automáticamente diagramas BPMN 2.0, tablas SIPOC, matrices RACI y documentación narrativa basada en Markdown.

Edita & Exporta

Modifica visualmente diagramas BPMN, edita narrativas en un editor WYSIWYG, y exporta a XML, PNG, SVG, CSV, Markdown, DOCX o PDF.

Espacios de Trabajo

Organiza proyectos con espacios de trabajo. Roles de Superadmin, Administrador, Editor y Visor con aislamiento de datos seguro.

Autohospedado & Seguro

Implementación en Docker con SQLite encriptado por SQLCipher. Fortalecido para cumplimiento de ISO 27001, HIPAA y GDPR con registros aprobados por el administrador.

Cómo Funciona

Cuatro sencillos pasos desde evidencia sin procesar hasta documentación accionable.

01

Sube Evidencia

Sube documentos de texto como POEs, notas de reuniones o correos que describan tus procesos.

02

Análisis de IA

ProcessAce usa tu LLM elegido para extraer pasos, actores, sistemas y relaciones de la evidencia.

03

Genera Artefactos

Obtén diagramas BPMN profesionales, tablas SIPOC/RACI y documentación narrativa autogenerada.

04

Edita & Exporta

Refina artefactos con el editor interactivo. Exporta como XML, PNG, SVG, CSV, Markdown, DOCX o PDF.

En funcionamiento en minutos

Solo necesitas Docker Compose. Trae tu propia clave API de LLM.

1. Clona el repositorio
git clone https://github.com/jgleiser/ProcessAce.git
cd ProcessAce
2. Configura el entorno
cp .env.example .env
# Edita .env y establece ENCRYPTION_KEY
3. Ejecuta con Docker Compose
docker compose up -d --build
4. Abre la Interfaz Web
# Navega a http://localhost:3000
# Registra tu primera cuenta (será Administrador)
# Configura el proveedor LLM en los Ajustes

Requisitos Previos

  • Docker & Docker Compose
  • Una clave API de LLM (OpenAI, Google GenAI o Anthropic)
  • Una cadena Hexadecimal de 32 bytes (para encriptación segura)
